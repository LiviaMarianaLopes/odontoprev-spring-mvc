Bem-vindo ao Sistema de Gerenciamento de Pacientes e Dentistas

Este sistema permite a gestão eficiente de pacientes e dentistas para a Odontoprev, com funcionalidades de cadastro, consulta, edição e exclusão de dados.

Como Funciona o Sistema?

O sistema é projetado para facilitar a interação entre pacientes, dentistas e a administração da Odontoprev. Com uma interface simples, você pode gerenciar dados e garantir a segurança dos processos.
